3. 수정 내용 상세

3.1 분절관 이음부 신축량 공식 표시 불일치 수정 (🔴 → 완료)

문제: 보고서(ReportPage.tsx)와 결과 탭(ResultPage.tsx)에서 이음부 신축량을 아래 간이공식으로 표시하고 있었으나, 실제 엔진은 더 엄밀한 지침 해설식 5.3.28~5.3.35를 사용한다.

[수정 전 — 보고서 표시 공식 (잘못됨)]
|uJ| = Uh × sin(π × Lj / L)
※ 이는 굽힘각 공식 θJ의 변형으로, 이음부 신축량과는 다른 식
[수정 후 — 실제 엔진 공식 (해설식 5.3.28~5.3.35)]
|uJ| = u0 × ūJ

u0 = α1 × Ua,  Ua = (1/2) × Uh

β1 = √(K1×Lj / (E×A)),  γ1 = π×Lj / (2×L')  [L'=2L]

ūJ = (sinh(β1γ1) − sin(β1γ1)) / (cosh(β1γ1) + cos(β1γ1))
※ 일본 수도시설 내진공법지침 원식 / 지침 부록C 해설식 5.3.28~5.3.35

수정 범위: src/pages/seismic-detail/ReportPage.tsx (라. 지진시의 이음새 신축량 섹션 전체 재작성), ResultPage.tsx (공식 레이블 수정)

3.3 부등침하 변형률 공식 주석 수정 (🟡 → 완료)

[수정 전 주석 (잘못된 설명)]
// ε_d = D_settle / (2 × L_settle)
// D_settle: 부등침하량 (m), L_settle: 침하 구간 길이 (m)
[수정 후 주석 (정확한 근거 기재)]
// 지침 해설식(5.3.41): ε_d = δ / (2 × l)
// 근거: tan(θ)/2 = (δ/l)/2 = δ/(2l) 로 직접 적용 (지침 부록C 예제값 검증 완료)
// ※ "δ²/(2l) 소변위 근사"가 아닌 지침 해설식(5.3.41) 그대로 적용

코드 로직은 변경 없음. 이전 주석이 "소변위 근사 δ²/(2l)"로 잘못 기재되어 있었으나 실제 코드와 지침 모두 δ/(2l) 사용.
